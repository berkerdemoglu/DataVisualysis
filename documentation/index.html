Required Modules

DataVisualysis uses some external (non-built-in) modules to perform some tasks:

  • matplotlib
  • numpy
  • requests

Module Installation

Copy and paste the following commands one by one to the command prompt:

pip install --user matplotlib pip install --user numpy pip install --user requests pip install --user mplcursors
