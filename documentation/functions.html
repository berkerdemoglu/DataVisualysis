DataModeler — The main class for analyzing and visualizing data

All following functions are called as object_name.function_name(arguments).

  • __init__(path=None, dataset=None)

    If path and dataset are both provided, then the array in the .json file will be used as the dataset. If neither are provided, then the default path stored in json_files/default_directory.json will be used as the path argument.

    @param path: The path to the .json file containing an array of numbers.
    @param dataset: A list of numbers to be used as the dataset.

    A StatisticsCalculator object is instantiated in the __init__ method. It can be used to calculate various statistics for a given dataset.

Statistical Functions

All statistical functions are printed to the terminal in show_results() and compare_datasets().
A DataModeler object instantiates a StatisticsCalculator object in its constructor method. Keep in mind that one can instantiate a StatisticsCalculator object by providing a dataset such as 'sc = StatisticsCalculator(dataset)'. In that case, call the following functions without 'calculate.'.

  • calculate.mean()

    Calculates and returns the arithmetic mean of the dataset.

  • calculate.median()

    Calculates and returns the median of the dataset.

  • calculate.mode(mode_occurrence=False)

    Calculates and returns the mode of the dataset.

    @param mode_occurrence: If there is no mode, returns the string, "The list the has no mode", regardless of mode_occurrence's value. If mode_occurrence is False, it will return only the mode. If it is True, it will return a list with 2 elements; the first being the mode and the second being how many times the mode occurs.

  • calculate.std()

    Calculates and returns the standard deviation of the dataset.

  • calculate.var()

    Calculates and returns the variance of the dataset.

  • calculate.percentile(percentile)

    Calculates and returns 'percentile'th percentile of the dataset.

  • calculate.occurrences()

    Returns a list containing how many times each number in the dataset occurs. The list contains tuples with the first element of the tuple being the number itself and second element being its number of occurrences.
